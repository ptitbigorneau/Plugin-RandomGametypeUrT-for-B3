# randomgametypeurt plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

randomgametypeurt B3 plugin (v1.0.1) for Urban Terror

Installation:

1. Place the randomgametypeurt.py in your ../b3/extplugins and the 
randomgametypeurt.ini in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="randomgametypeurt" config="@b3/extplugins/conf/randomgametypeurt.ini"/>
